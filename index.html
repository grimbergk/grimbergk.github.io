A Sample Web Page

Welcome to my very first web page! This page was created as a part of my Intro to Web Development course at ABC University.

Here are some things I learned in this course:

  • HTML stands for Hyper Text Markup Language
  • CSS stands for Cascading Style Sheets
  • Web pages are built using a combination of HTML and CSS

Here's an example of some HTML code:

<!DOCTYPE html>
  <html>
    <head>
      <title>My Web Page</title>
    </head>
    <body>
      <h1>Welcome to my web page!</h1>
      <p>This is some text on my page.</p>
    </body>
  </html>

And here's an example of some CSS code:

body {
    background-color: #f2f2f2;
    font-family: Arial, sans-serif;
  }
  
  h1 {
    color: #333333;
    text-align: center;
  }

Thanks for visiting my web page!
